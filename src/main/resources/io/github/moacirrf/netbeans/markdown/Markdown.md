# Markdown for Netbeans ![Description Here](https://netbeans.apache.org/images/nblogo48x48.png)

***

## Description
This plugin adds some additional features to Apache Netbeans Markdown Editor.
- Preview
- Split Window
- Suggestion
- Export to DOCX, PDF and HTML
